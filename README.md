# GAS

Mit GAS kann man ganz einfach einen Stundenplan anhand von gewählten Kursen erstellen lassen.

# Was ist GAS?

GAS steht für „Genetic Algorithm Scheduler“. GAS hat den speziellen Anwendungsfall der Stundenplanerstellung in der Oberstufe. Da SuS individuell Kurse (nach einem bestimmten Schema) wählen können, treffen die Organisatoren auf das Problem, einen Stundenplan für die SuS zu erstellen. GAS soll bei dem Problem helfen und die Planung (teilweise) übernehmen und per Knopfdruck einen Stundenplan erstellen.

# Wie arbeitet GAS?

GAS arbeitet mit genetischen Algorithmen. Diese kann man sich vorstellen wie eine Evolution. 

Dabei werden "Start"-Stundenpläne erstellt und diese jeweils leicht verändert, gekreuzt und intelligent, aber dennoch zufällig angepasst. So entsteht Diversität und die Stundenpläne können sich durch sinnvolle Selektion in eine bestimmte Richtung entwickeln.

# Wichtige Fragen und Anmerkungen:

### Wie installiere ich GAS?
GAS läuft nur auf Windows-PCs. Es muss zunächst die neueste Version rechts bei *Releases* ausgewählt werden, dann auf die *.zip*-Datei geklickt werden. Diese herunterladen, im Dateiexplorer entpacken (*Rechtsklick > Alle extrahieren*) und *setup.exe* ausführen. Kommt eine Windows-Sicherheitswarnung, auf *Weitere Informationen* und dann *Trotzdem ausführen* klicken. GAS wird jetzt installiert. Eventuell muss dazu noch .NET heruntergeladen werden, einfach akzeptieren.

Alternativ zur Installation kann auch direkt nach dem Entpacken *GAS.exe* gestartet werden.

### Kann man von einem anderen Programm Daten importieren?
Ja, von winprosa. In der Hilfe ist dies beschrieben.

### Speichert GAS die Daten ab?
Nein, GAS macht dies nur, wenn explizit *Datei > Daten exportieren* ausgewählt wird.

### Ersetzt GAS ein Verwaltungsprogramm?
GAS ist keine Verwaltungssoftware. Zur Oberstufenverwaltung sollte weiterhin winprosa oder ähnliches verwendet werden. GAS sollte nicht als Ersatz anerkannt werden, sondern viel mehr als eine Erweiterung.

### Findet GAS immer eine Lösung?
GAS läuft mit dem Zufallsprinzip. Obwohl der Zufall nicht blind ist, kann nicht garantiert werden, dass eine Lösung gefunden wird. GAS ergänzt die Arbeit von Hand und ist eine Unterstützung. Auch kann das Finden einer Lösung sehr lange dauern. Geduld ist hier gefragt und die Computer sollten über die Möglichkeit verfügen, über Nacht zu laufen, sodass kein Datenverlust riskiert wird. Das finden einer Lösung wird damit explizit nicht garantiert.

### Können Stundenpläne für mehrere Halbjahre gleichzeitig erstellt werden?
GAS kann nicht die Stundenpläne für mehrere Halbjahre erstellen. Dazu müssen jeweils die Daten angepasst werden und dann kann ein neuer Stundenplan berechnet werden.

# Noch Fragen oder Anmerkungen?

Im Programm ist eine Hilfe-Datei mit eingebunden, diese kann durch *Über > Hilfe* erreicht werden.
